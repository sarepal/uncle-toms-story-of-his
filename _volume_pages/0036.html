{% raw %}In sharp contrast with the experience I had felt of{% endraw %}
{% raw %}the contempt and brutality of my earthly master,{% endraw %}
{% raw %}I basked, as it were, in the benign smiles of this{% endraw %}
{% raw %}Heavenly Being. I thought, "He'll be my dear refuge{% endraw %}
{% raw %}—He'll wipe away all tears from my eyes." " Now{% endraw %}
{% raw %}I can bear all things ; nothing will seem hard after{% endraw %}
{% raw %}this." I felt sure that if " Massa Riley " only knew{% endraw %}
{% raw %}Him, he would not live such a coarse, wicked, cruel{% endraw %}
{% raw %}life. Swallowed up in the beauty of the divine love,{% endraw %}
{% raw %}I " loved my enemies, and prayed for them that did{% endraw %}
{% raw %}despitefully use and entreat me."{% endraw %}

{% raw %}mind as I went home, I became so excited that I{% endraw %}
{% raw %}turned aside from the road into the woods, and{% endraw %}
{% raw %}prayed to God for light and for aid with an earnest¬{% endraw %}
{% raw %}ness, which, however unenlightened, was at least{% endraw %}
{% raw %}sincere and heartfelt; and which the subsequent{% endraw %}
