{% raw %}NEGRO CONTROVERSY{% endraw %}
{% raw %}IN CANADIAN TOWN{% endraw %}
{% raw %}FROM OUR ONTARIO CORRESPONDENT{% endraw %}
{% raw %}The Negro population of the small{% endraw %}
{% raw %}Ontario town of Dresden last Saturday{% endraw %}
{% raw %}celebrated the seventy-third anniversary{% endraw %}
{% raw %}of the death of Uncle Tom whom{% endraw %}
{% raw %}Harriet Beecher Stowe immortalized in{% endraw %}
{% raw %}her novel Uncle Tom's Cabin. Uncle{% endraw %}
{% raw %}Tom's real name was the Rev. Josiah{% endraw %}
{% raw %}Henson, and he lived in Dresden as an{% endraw %}
{% raw %}escaped slave for more than 50 years.{% endraw %}
{% raw %}It was from the town that he directed{% endraw %}
{% raw %}immigration of freed slaves to a colony set{% endraw %}
{% raw %}up on a 300-acre site about a mile west of{% endraw %}
{% raw %}the town. Mostly through his efforts to{% endraw %}
{% raw %}rehabilitate slaves (he made three visits .to{% endraw %}
{% raw %}Britain to lecture and to collect funds, and{% endraw %}
{% raw %}was presented to Queen Victoria in 1877){% endraw %}
{% raw %}Dresden became one of the shrines of{% endraw %}
{% raw %}Negro freedom in North America.{% endraw %}
{% raw %}To-day, by a strange twist of fate,{% endraw %}
{% raw %}Dresden is the centre of the most ardent{% endraw %}
{% raw %}racial discrimination battle in Canada. Since{% endraw %}
{% raw %}the end of the war Negro residents have{% endraw %}
{% raw %}been refused service in certain restaurants,{% endraw %}
{% raw %}barber shops, and stores. There has been{% endraw %}
{% raw %}no violence, but an atmosphere of restrained{% endraw %}
{% raw %}ill feeling has existed, which the older{% endraw %}
{% raw %}Negroes have taken philosophically.{% endraw %}
{% raw %}ASSOCIATION FORMED{% endraw %}
{% raw %}The younger members of the 350 Negroes{% endraw %}
{% raw %}in the colony have not been quite so tem¬{% endraw %}
{% raw %}perate. Returning service men took par¬{% endraw %}
{% raw %}ticular objection to this discriminatory treat¬{% endraw %}
{% raw %}ment, and formed an association to protect{% endraw %}
{% raw %}their rights. Delegations were sent to the{% endraw %}
{% raw %}Premier, Mr. Frost, who initiated legisla¬{% endraw %}
{% raw %}tion which was accepted by the Ontario{% endraw %}
{% raw %}House.{% endraw %}
{% raw %}In spite of this, cases of discrimination{% endraw %}
{% raw %}continued. Testing the Act last November,{% endraw %}
{% raw %}two Negroes were told to leave a pafe" in the{% endraw %}
{% raw %}main street of the town, the owner refusing{% endraw %}
{% raw %}to serve them with coffee. In February of{% endraw %}
{% raw %}this year this case was tried and a conviction{% endraw %}
{% raw %}obtained, the first since the trouble started.{% endraw %}
{% raw %}Since then things have been fairly quiet{% endraw %}
{% raw %}in Dresden. Government officials say that{% endraw %}
{% raw %}if the few fervid segregationalists remain{% endraw %}
{% raw %}passive the trouble might end. However,{% endraw %}
{% raw %}one restaurant owner in particular insists{% endraw %}
{% raw %}that he will refuse to serve coloured people.{% endraw %}
{% raw %}An interesting point to the story is that{% endraw %}
{% raw %}there is little in Dresden to warrant a colour{% endraw %}
{% raw %}bar. The Negro population are not poor.{% endraw %}
{% raw %}Their children mix at school with whites.{% endraw %}
{% raw %}Negroes work on farms, and in the small{% endraw %}
{% raw %}industries of the place, beside whites.{% endraw %}
{% raw %}SERIES OF CAMPAIGNS{% endraw %}
{% raw %}The influence of the Dresden case, as it{% endraw %}
{% raw %}has become known, has spread far beyond{% endraw %}
{% raw %}the confines of the town. Coupled with{% endraw %}
{% raw %}attempts to discriminate in other parts of{% endraw %}
{% raw %}the country, notably in Hamilton, Toronto,{% endraw %}
{% raw %}and Windsor, it has brought about a series{% endraw %}
{% raw %}of campaigns to better the lot of the{% endraw %}
{% raw %}coloured people of Canada.{% endraw %}
{% raw %}One of the most important has been an{% endraw %}
{% raw %}attempt to influence the Federal Govern¬{% endraw %}
{% raw %}ment to change its Immigration Act, which{% endraw %}
{% raw %}limits the number of Negro people entering{% endraw %}
{% raw %}the country whether of British or of foreign{% endraw %}
{% raw %}nationality. At present people from Paki¬{% endraw %}
{% raw %}stan, the West Indies, India, and other such{% endraw %}
{% raw %}British countries are denied entry, except in{% endraw %}
{% raw %}special circumstances, because the Govern1{% endraw %}
{% raw %}ment maintains that they have difficulty in{% endraw %}
{% raw %}adjusting themselves to the Canadian life{% endraw %}
{% raw %}and climate. A recent delegation from the{% endraw %}
{% raw %}Negro Citizenship Association of Toronto{% endraw %}
{% raw %}presenting a brief to the Minister of Immi¬{% endraw %}
{% raw %}gration, Mr. Harris, urged the Government{% endraw %}
{% raw %}to stop the " dangerous and discriminatory{% endraw %}
{% raw %}business of creating two classes of her{% endraw %}
{% raw %}Majesty's subjects."{% endraw %}
{% raw %}Other Negro associations have been press¬{% endraw %}
{% raw %}ing for adequate anti-discrimination laws,{% endraw %}
{% raw %}and for the prosecution of alleged attempts{% endraw %}
{% raw %}at intimidation of coloured people. News¬{% endraw %}
{% raw %}papers and religious leaders have supported{% endraw %}
