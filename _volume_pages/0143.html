{% raw %}On returning to England the exhibition was still{% endraw %}
{% raw %}in progress. There seemed no diminution of the{% endraw %}
{% raw %}crowd. Like the waters of the great Mississippi,{% endraw %}
{% raw %}the channel was still full, though the individuals{% endraw %}
{% raw %}were changed.{% endraw %}
{% raw %}But among all the exhibitors from every nation in{% endraw %}
{% raw %}Europe, and from Asia, America, and the Isles of the{% endraw %}
{% raw %}Sea, there was not a single black man but myself.{% endraw %}
{% raw %}There were negroes there from Africa, brought to{% endraw %}
{% raw %}be exhibited, but no negro-exhibitors but myself.{% endraw %}
{% raw %}Though my condition was wonderfully changed from{% endraw %}
{% raw %}what it was in my childhood and youth, yet it was a{% endraw %}
{% raw %}little saddening to reflect that my people were not{% endraw %}
{% raw %}more largely represented there. The time will yet{% endraw %}
{% raw %}come, I trust, when such a state of things will no{% endraw %}
{% raw %}longer exist.{% endraw %}
